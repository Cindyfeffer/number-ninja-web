Number Ninja web version

This package contains a safe browser game:
- Open index.html in any web browser
- No installation required
- No executable installers are included

To share by email, send the ZIP file:
- NumberNinjaWeb.zip

If you want a clickable link, upload the folder or zip to a trusted host
(e.g. GitHub Pages, Netlify, Vercel, or Dropbox public link).

This version uses only HTML, CSS, and JavaScript, which is much safer than
sending an app bundle or installer.
